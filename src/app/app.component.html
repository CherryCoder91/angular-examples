Routing in Angular

In Angular Routing is handled by the Framework. Routing configuration is configured within routing modules which are coupled with a routed module which typically expresses a page. Routes can be configured to be statically loaded as part of the app or defined in a way that will support lazy loading of that route and the associated assets of that routed module. Routes can also contain parameters and certain routes can reasoned about as to whether the user should be allowed to access them.
